Privacy Policy – Posisi Dimana

Aplikasi: Posisi Dimana
Pengembang: Loods 4.C.a.7.5

1. Pengumpulan Data

Aplikasi ini hanya mengakses lokasi perangkat (koordinat GPS) saat Anda memilih untuk membuat tautan lokasi atau kode QR. Aplikasi tidak mengumpulkan nama, email, nomor telepon, atau data identitas lainnya.

2. Tujuan Penggunaan Data

Data lokasi digunakan hanya untuk:

  • Menampilkan akurasi lokasi.
  • Menghasilkan URL lokasi yang dapat dibagikan.
  • Membuat kode QR yang mewakili lokasi untuk dibagikan.

Aplikasi tidak mengunggah lokasi ke server atau membagikannya ke pihak ketiga.

3. Pembagian Data

Tidak ada data yang dibagikan ke pihak ketiga. Aplikasi tidak menampilkan iklan atau menggunakan layanan analitik eksternal.

4. Penyimpanan dan Keamanan

Data lokasi hanya disimpan sementara pada perangkat untuk kebutuhan pembuatan URL/QR dan tidak disimpan secara permanen atau dikirim ke server.

5. Hak Pengguna

Karena aplikasi tidak menyimpan data pribadi, tidak ada data yang harus diminta untuk dihapus. Untuk pertanyaan lebih lanjut, hubungi pengembang.
